# ⌚ Neurolex Wearable App

This directory contains the source code for the Neurolex smartwatch application.

## 📱 Supported Platforms
-   **WearOS (Android):** Built with Kotlin/Jetpack Compose.
-   **watchOS (Apple):** Built with SwiftUI.
-   **Cross-Platform:** Flutter implementation for broader compatibility.

## 🚀 Features
-   **Real-time Sensor Access:** Reads Heart Rate (HR) and GSR/EDA raw data directly from hardware.
-   **Haptic Engine:** Delivers custom vibration patterns for "Focus" and "Stress" alerts.
-   **Biofeedback UI:** Displays breathing animations on the wrist.
-   **Sync:** Background sync with the Neurolex Mobile App & Backend.

## 🛠️ Tech Stack
-   **Language:** Dart (Flutter) / Kotlin / Swift
-   **Sensors:** Accelerated Sensor capability for HRV calculation.
